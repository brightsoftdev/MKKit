UIView(MKMetrics) Category Reference

Declared in UIView+MKMetrics.h

Overview

Overview

The UIView+MKMethrics catagory adds methods to the UIView class to support the use of MKMetrics when laying out subviews.

Tasks

Layout

  • - layoutForMetrics: A specific layout call for UIView instances to use.

    Note This method will not be automatically called. When using you should call it when the interface orientation changes.

Layout Settings

  • - setSize:forMetrics: Sets a size for the given metrics.
  • - setLocation:forMetrics: Sets a location for the given metrics. The location point corisponds the top left corner of the view.
  • - setRect:forMetrics: Sets a rect for the given metrics.

Reference

  • portraitRect Used by MKMetrics when the layoutSubview:forMetrics: is called
  • landscapeRect Used by MKMetrics when the layoutSubview:forMetrics: is called

Memory

  • - clearMetrics Cleans up after the a subview is layed out. You do not need to call this method directly.
