Inherits from MKControl : UIControl
Declared in MKBarButtonItem.h

Overview

MKBarButtonItem creates buttons for use on a tab bar. The button that will be displayed depends on the MKBarButtonItemType that is set. Currently There are three types:

  • MKBarButtonItemIcon: a button created from an image. Use initWithIcon method to for this type.
  • MKBarButtonItemBackArrow: a triangle pointing to the left.
  • MKBarButtonItemForwardArrow: a triangle pointing to the right.
