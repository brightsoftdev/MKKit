Overview

MKPropmtView displays an information prompt on the top of the screen. There are three types of prompts that you can pick to display, depending on the importance of the message.

  • MKPromptTypeGreen: a prompt with a green background.
  • MKPromptTypeAmber: a prompt with a yellow background.
  • MKPromptTypeRed: a prompt with a red backgrount.

MKPromptView will automaticaly adjust its height to fit the given message.

Tasks

Creating

  • - initWithType:title:message: Returns a new instance of MKPromptView.
  • + promptWithType:title:message:duration: Creates and displays an instance of MKPromptView. The view is displayed with the MKViewAnimationTypeMoveInFromTop.

Behavior Properties

  • type The type of prompt that is displayed
  • duration The Time in Seconds until the prompt is dismissed. Set to zero to disable.

Drawing Methods

  • - frameForMessage: Returns the frame for the view that will fit the message text. This is an internal method and should not be called directly.
  • - frameForText: Returns the frame for the UILabel that will fit the message text. This is an internal method and should not be called directly.
