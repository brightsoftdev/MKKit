MKSwipeCellView Class Reference

Inherits from MKView : UIView
Declared in MKSwipeCellView.h

Overview

MKSwipeCellView provides a view to be displayed when a cell is swiped. The view uses MKSwipeCellItems to provide control buttons.

Tasks

Creating

  • - initWithItems:cell: Returns an instance of MKSwipeCellView.

Showing and Removing

  • - show Shows the veiw on the cell. The view is animated onto the cell by sliding in from the right side of the screen.
  • - remove Removes the view from the cell. The view is animated of the cell by sliding out to the right side of the screen
