Inherits from MKTableCell : UITableViewCell

Overview

The MKTableCellSwitch class creates a table cell with an UISwitch on the right hand side. When the switch position is changed it calls the valueDidChange:forKey: delegate method.

Note The switches isOn value is converted to an NSNumber object before the delegate method is called.
