MKCoreData Class Reference

Inherits from NSObject
Declared in MKCoreData.h

Overview

The MKCoreData class is a singleton instance used to work with Core Data thoughout an application. A NSManagedObject context needs to set before this object can preform any methods. You do this by calling sharedDataWithContext: 

[MKCoreData sharedDataWithContext:myContext];

  • + sharedDataWithContext: Creates the singleton object on sets the default managed object context.
