Inherits from MKControl : UIControl

Overview

The MKCheckBox class creates a 30x30 checkbox object. When the user touches the check box it is toggled on and off.

You should use initWithFrame: to initalize this class.

MKCheckBox send a UIControlEventValueChanged action.
