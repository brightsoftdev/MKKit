Overview

Overview

MKTableCellSlider creates a table with a UISlider placed on it for a control.

Observing Actions

This class will send messages to the valueDidChange:forKey: delegate method when the slider is moved.

Note The value of the slider is passed through the delegate as an instance of NSNumber.
