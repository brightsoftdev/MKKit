Overview

The MKControl class is the super class of all other control objects. This class implements the MKControlDelegate protocol.

completedAction:

Parameters

actionBlock
the code block.

Discussion

Sets a code block to complete after an action is completed.

The action block is in this format ^(MKAction action). MKAction is the action that was completed.

The action will be on of the following types:

  • MKActionTouchDown: a touch down action.
  • MKActionTouchUp: a touch up action.
