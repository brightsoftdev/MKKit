MKRandoms Class Reference

Inherits from NSObject
Declared in MKRandoms.h

Overview

The MKRandoms object preforms methods that return random values. It can work with arrays or just integes to get random values.

Tasks

Randoms with Arrays

  • - randomFromArray:repeat: Returns a random object from a given array.
  • - randomArrayWithArray: Returns an array resorted in a random order.

Randoms with Integers

  • - randomNumberOutOf: Returns and random integer between 0 and a given number.
