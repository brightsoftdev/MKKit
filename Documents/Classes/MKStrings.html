Inherits from MKObject : NSObject

Overview

The MKStrings class creates specaillty strings such as number and date formats, paths, and UUIDs. You have the choice to use instance methods or class methods for all strings.

stringWithDate:format:

Parameters

date
The date to format.
format
a string representation of the date format.

Discussion

Returns a string representation from a date and a specified format.

Here are some examples

Date Formats:

  • @"MMM dd YYYY" will return Jan 01 2011.
  • @"MMMM dd YYYY" will return January 01 2011.
  • @"MM-dd-YYYY" will return 01-01-2011.

Time Formats:

  • @"h:mm" will return 1:00.
  • @"hh:mm.ss" will return 01:00.00.
  • @"kk:mm" will return 13:00.
