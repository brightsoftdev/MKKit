Inherits from MKTableCell : UITableViewCell

Overview

The MKTableCellSegmented class creates a table cell with a UISegmentedController filling the cell. Set the items of the cell by Segmented Control by setting the segmentItems property.

When the selected segment changes the valueDidChange:forKey: delegate method is called.

Note The selectedSegmentIndex property is converted to a NSNumber object before the delegate method is called.
