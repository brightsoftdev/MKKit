Overview

Overview

MKHTMLNode provides access to the nodes of an web site. There are several methods to navigate through the nodes and obtain information about the node.

Creating Instances

MKHTMLParser works with MKHTMLNode to create instances of MKHTMLNode. Using the MKHTMLParser is the prefered method for creating the first instance. After that use the Finding Nodes method of MKHTMLNode.

Tasks

Creating Instances

  • - initWithNode: Creates an instance of MKHTMLNode. This method should not need to be called directly.

Getting Text

  • - allText Returns all the text in the node to include the text of all the children.
  • - allTextForNodesNamed: Returns all the text from the first node with the specified name.
  • - htmlString Returns the html code of the node in a string format.

Finding Nodes

  • - nodeWithAttribute:value:allowPartial: Finds the first node has a given attribute matching the specified value.
  • - childrenWithAttribute:value:allowPartial: Finds all nodes that have a given attribute matching the specified value.
  • - childNodeNamed: Finds the first node of the given name.
  • - childrenNamed: Finds all the children nodes of the given name.
  • - children Finds all the children of the node.
  • - nextChildNode Moves to the next child of the tree.
  • - previousChildNode Moves to the previous node of the tree.
  • - parentNode Moves to the parent Node of the tree.

Node Attributes

  • - nodeName Returns the name of the current node.

Getting Node Types

  • - nodeType Returns the type of the node. MKHTMLNode supports the following types:
    • MKHTMLNodeDiv: node with <div> name.
    • MKHTMLNodeP: node with <p> name.
    • MKHTMLNodeScript: node with <script> name.
    • MKHTMLNodeA: node with <a> name.
    • MKHTMLClosingNode: node that closes out another node </somenode>.
    • MKHTMLNodeUnknown: node that cannot be identified by one of the other types.

childrenWithAttribute:value:allowPartial:

Finds all nodes that have a given attribute matching the specified value.

- (NSArray *)childrenWithAttribute:(NSString *)attribute value:(NSString *)value allowPartial:(BOOL)partial

Parameters

value
The value the attribute should have.
partial
YES if a partial match to the value is allow. NO if an exact match is needed.
attruibute
The attribute to look for.

Return Value

NSArray instance populated with MKHTMLNode objects.

Discussion

Finds all nodes that have a given attribute matching the specified value.

For exampe to find the node <div class="article">, call [someNode childrenWithAttribute:@"class" value:@"article" allowPartial:YES].
