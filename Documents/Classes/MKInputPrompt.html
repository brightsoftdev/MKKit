Overview

The MKInputPrompt class provides a pop-over text input prompt. The input prompt includes space for a message to the user. MKInputPrompt will automatically adjust it's hieght to fit the entire message.

Tasks

Creating

  • - initWithMessage: Returns and initalized instance of MKInputPrompt. When this method is use it is your responsiblity to display/remove the prompt, and handel the inputed text. For automatic display and input handeling use showWithMessage:onDone:.
