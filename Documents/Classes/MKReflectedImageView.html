Overview

The MKReflecedImageView class displays an image and creates a reflection of that image underneath it.

The frame for this view is calcuated by using the demtions of the provided image. Please use correct image sizes. The view will be the same width of the image, and height is 25% larger plus 2 pixs. For example if your imgae is 100px high the height of the view will be 127px (100*.25 + 2).
