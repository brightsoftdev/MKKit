MKMath Class Reference

Inherits from MKObject : NSObject
Declared in MKMath.h

Overview

The MKMath object preforms simple math on the contents of an Array. These methods are useful for calculating trasient core data attributes.

Note Arrays passed to this object must contain NSDecimalNumber objects.
