MKIAPDelegate Protocol Reference

Conforms to NSObject
Declared in MKIAPController.h

Overview

The MKIAPDelegate defines methods to recieve data from inApp Purchase requests. The delegate needs to be used only if you are using the instace methods of MKIAPController. The factory methods will not call methods on the delegate.

Tasks

  • - didRecieveResponse: Called when a response is recieved by the MKIAPController.
  • - didCompleteTransaction: Called when a transaction is completed by the MKIAPController.
  • - didRestorePurchases Called when purchase are restored.
  • - didError: Called if there is an error during a request.
