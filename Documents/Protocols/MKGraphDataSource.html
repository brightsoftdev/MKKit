MKGraphDataSource Protocol Reference

Conforms to NSObject
Declared in MKGraphDataSource.h

Overview

Provides the data for a MKGraphView. All methods are required for the graph view to work correcly.

Tasks

Set Up

  • - numberOfDataSetsForGraphView: Return the nuber of data sets that your graph will use. required method
  • - scaleForGraphView: Retrun the scale of the graph. Scales can be made using the MKGraphScaleMake(float max, float min) funtion. required method

Providing Data

  • - graphView:dataSetForIndex: Return a MKGraphDataSet for each set of data in the graph. required method

Instance Methods

Return a MKGraphDataSet for each set of data in the graph.
