MKInputValidation Protocol Reference

Conforms to NSObject
Declared in MKInputValidation.h

Overview

The MKInputValidation protocol gives methods for objects to validate user input. Objects that use implement the MKInputValidation protocol can use different MKValidationTypes to tell the MKValidator which methods to call.

These types are:

  • MKValidationNone: No valition type set.
  • MKValidateIsaNumber: Calls the inputIsaNumber: method.
  • MKValidateHasLength: Calls the inputHasLength: method.

Tasks

Text Input Validation Methods

  • - inputIsaNumber: Retuns YES if the text is a number.
  • - inputHasLength: Returns YES if text has a length greater than 0.
