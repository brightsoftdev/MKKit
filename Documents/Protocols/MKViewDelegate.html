Overview

The MKViewDelegate protocol provides delegate methods for MKView and its subclasses

Add/Remove methods

  • - shouldRemoveView: Some MKView objects will automatically remove themselfs after a certain action. This method can be used to set if the view should be removed or not. If this method is not used the MKView object follows its default behaivor.
