MKViewControllerDelegate Protocol Reference

Conforms to NSObject
Declared in MKViewControllerDelegate.h

Overview

The MKViewControllerDelegate protocol gives methods to observe actions from a MKViewConrtroller.

Note not all delegate methods are called by all MKViewController sub classes. Read over this documention of the subclasses to find what methods are called.

Tasks

View Controller Methods

  • - viewControllerIsDone: Called the done button of a MKViewController is tapped. This is typically used to dismiss a model view controller
  • - viewControllerAction: Called when the action button of MKViewController is tapped.

Login View Controller Methods

  • - pinIsValidated: Called when a PIN is entered into the MKLoginViewController.
  • - setPin: Called when a New Pin has been set.
  • - maxPinAtemptsMade Called when the maximum PIN Atempts have been made.
